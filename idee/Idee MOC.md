## LevelUp IRL (a.k.a. LifeGuild)

**Stato:** Concept dettagliato con pitch deck strutturato

App di gamification della vita reale. Trasforma obiettivi quotidiani in missioni con sistema di progressione e Battle Pass.

### Core Loop

| Categoria | Tipo missioni |
|-----------|--------------|
| **Corpo** | Allenamento, sonno, idratazione |
| **Mente** | Lettura, studio, meditazione |
| **Social** | Connessioni, uscite, azioni collaborative |

- Le attività cooperative danno **bonus punti** (incentivo alla connessione reale)
- Niente guilds nella versione finale
- Integrazione health data: **HealthKit** (iOS) / **Google Fit** (Android)
- Focus mode: **Pomodoro + screen blocking**

### Monetizzazione

**Battle Pass** — €4.99/mese
- Missioni premium sbloccabili
- Skin ed effetti visivi
- Contenuto esclusivo stagionale

### Stack tecnico

```
Frontend:  Flutter o React Native
Backend:   Supabase o Firebase
Health:    HealthKit / Google Fit
```

### Struttura Pitch Deck

1. Hook — problema (vita reale vs giochi)
2. Soluzione — gamification daily life
3. Demo core loop
4. Mercato target
5. Modello di ricavo (Battle Pass)
6. Roadmap
7. CTA investitori / early adopters
